User Manual

Web Desktop is next generation remote desktop tool based on HTML5, you can access remote desktop provided by data center anytime anywhere as long as having a browser compatible HTML5. 1. Access remote desktop It is easy to access remote desktop. The thing that you need to do is click the icon at web desktop.  As you can see, it is very similar to using Microsoft RDP client to connect a remote desktop. Anyway, there are some different, 1) Web Desktop connect to a remote desktop although browser, not RDP client. 2) You need Not care the IP address, username and password of remote desktop, and where the remote desktop is at as well. 2. Access a specify application instead of whole desktop. System administrator can deliver kinds of application. The application will be appeared at web desktop once it is delivered. When click an application icon, you will get a running of the application.
